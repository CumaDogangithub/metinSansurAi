Yerel Kurulum

Bu uygulama tamamen sizin makinenizde çalışır — ne kişisel veri, ne de örnek metinler dış sunuculara gönderilir. Üç bileşene ihtiyaç vardır.

  1. 1
    Python 3.10+ — sistem genelinde kurulu olması yeterli. 
    python --version  # 3.10 veya üzeri
  2. 2
    Ollama — yerel LLM çalışma ortamı. ollama.com/download adresinden indirip kurun. Kurulduktan sonra arka planda otomatik servis verir (http://127.0.0.1:11434).
  3. 3
    Modeli indirin — varsayılan olarak {{ model_name }} kullanılır (~9 GB). Disk alanı yetmiyorsa qwen2.5:7b da çalışır ama doğruluk düşer. 
    ollama pull {{ model_name }}
  4. 4
    Python bağımlılıklarını kurun 
    pip install flask flask-cors ollama easyocr opencv-python pillow numpy requests

    EasyOCR ilk import'ta OCR modellerini otomatik indirir (~100 MB).

  5. 5
    Sunucuyu başlatın 
    python app.py

    Tarayıcıda http://127.0.0.1:5000

Sistem Gereksinimleri

💾 RAM

Minimum 16 GB önerilir (qwen2.5:14b için). 7B model 8 GB ile çalışır.

🖥️ İşlemci

CPU yeterli; ilk istek 6-10 sn, sonrakiler ~2-3 sn. CUDA varsa EasyOCR otomatik kullanır.

💿 Disk

Ollama + qwen2.5:14b ~10 GB. EasyOCR modelleri ek ~100 MB.

🌐 İnternet

Sadece kurulum sırasında (model indirme). Sonrasında tamamen offline çalışır.

Docker / Uzak Ollama

Eğer Ollama'yı başka bir adreste (örn. Docker compose servisi veya uzak sunucu) çalıştırıyorsanız OLLAMA_HOST ortam değişkenini set edin:

# Docker compose
OLLAMA_HOST=http://ollama:11434

# Uzak makine
OLLAMA_HOST=http://192.168.1.50:11434

# Linux/macOS (geçici)
OLLAMA_HOST=http://ollama:11434 python app.py

# Windows PowerShell (geçici)
$env:OLLAMA_HOST = "http://ollama:11434"; python app.py

Uygulama bu adresi sağlık kontrolü, modeller çağırma ve hata mesajları dahil her yerde kullanır. Set edilmemişse varsayılan: http://127.0.0.1:11434

Sorun Giderme

  • "Ollama kapalı" hatası — terminalden ollama serve komutu çalıştırın veya Ollama uygulamasını başlatın.
  • İlk istek 1 dakika sürdü — normal: model RAM'e yükleniyor. keep_alive=30m sayesinde sonraki istekler 2-3 sn.
  • JSON hata mesajı — model bağlamı tam anlayamadı; metni kısaltıp tekrar deneyin veya prompts.py'da MODEL_NAME'i daha güçlü bir modele alın.
  • Resim yüklemede yavaşlık — EasyOCR ilk yüklemede modelleri indiriyor; bir sonraki sefer hızlı olacaktır.
